# Cómo ver Doctoralia

La información de Doctoralia se ha separado del CRM de recepción.

- CRM recepción: `frontend/crm.html`
- Panel privado Doctoralia: `frontend/doctoralia-admin.html`

Para verlo en el dominio, sube la carpeta `frontend` y abre:

```txt
https://tu-dominio.com/doctoralia-admin.html
```

La pantalla tiene modo demo visual sin credenciales y modo real cuando el backend Node esté desplegado con `.env` configurado.

Importante: que una página no esté enlazada no es seguridad real. Para producción debe protegerse con contraseña, área privada o autenticación del hosting.
