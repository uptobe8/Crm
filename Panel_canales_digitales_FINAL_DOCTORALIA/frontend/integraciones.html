APIs necesarias

WhatsApp Cloud API

Para leads y conversaciones entrantes desde WhatsApp.

POST /webhook/whatsapp/lead

Gmail API

Para leads, respuestas y seguimiento comercial por email.

POST /webhook/gmail/lead

Web / Formularios

Para landings, formularios, eventos UTM y conversiones web.

POST /webhook/web/lead

HubSpot CRM API

Para contactos, deals, presupuestos, ventas y atribución.

GET /crm/deals?channel=canal

GA4 / Analítica

Para tráfico, sesiones, eventos, fuente/medio y conversiones.

POST /analytics/channel-report

Ads APIs

Para inversión, clics, impresiones, campañas y coste real.

GET /ads/spend?channel=canal
